HGCalML
===============================================================================

Requirements
  * DeepJetCore 2.3 (``https://github.com/DL4Jets/DeepJetCore``)
  * DeepJetCore 2.3 container (or latest version in general)
  
For CERN, a script to start the latest container in interactive mode can be found here:

``/eos/home-j/jkiesele/singularity/run_deepjetcore2.sh``


Setup
===========

```
git clone https://github.com/cms-pepr/HGCalML
cd HGCalML
source env.sh #every time
./setup.sh #just once
```
